# Projects: Mobile Security

Hands-on projects for understanding mobile application and device security.

---

## Projects Included

### 1. App Permission Analyzer (`app_permission_analyzer.py`)
Simulates analyzing mobile app permission requests:
- Identify overprivileged apps
- Detect permission abuse
- Learn about Android/iOS permission models
- Privacy risk scoring

**How to run:**
```bash
python app_permission_analyzer.py
```

### 2. Secure Storage Demo (`secure_storage_demo.py`)
Demonstrates secure vs insecure mobile data storage:
- Plaintext vs encrypted storage
- Keychain/Keystore simulation
- Insecure data location detection
- Data extraction simulation

**How to run:**
```bash
python secure_storage_demo.py
```

### 3. Mobile Threat Simulator (`mobile_threat_sim.py`)
Simulates common mobile security threats:
- Malicious app behavior
- Network interception
- Smishing detection
- Device compromise indicators

**How to run:**
```bash
python mobile_threat_sim.py
```

---

## Learning Objectives

- Understand mobile permission models and privacy risks
- Learn the difference between secure and insecure data storage
- Recognize common mobile threat patterns

## Requirements

- Python 3.x (standard library only)
