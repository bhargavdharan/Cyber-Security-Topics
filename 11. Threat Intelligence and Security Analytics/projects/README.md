# Projects: Threat Intelligence and Security Analytics

Hands-on tools for learning threat intelligence and security data analysis.

---

## Projects Included

### 1. IOC Analyzer (`ioc_analyzer.py`)
Analyze and classify Indicators of Compromise:
- Validate IP addresses, domains, and file hashes
- Check against known threat patterns
- Generate IOC reports
- Convert between IOC formats

**How to run:**
```bash
python ioc_analyzer.py
```

### 2. Log Correlator (`log_correlator.py`)
Correlates events across multiple log sources:
- Multi-source log aggregation
- Time-based correlation
- Attack chain reconstruction
- Alert generation

**How to run:**
```bash
python log_correlator.py
```

### 3. Threat Hunting Simulator (`threat_hunting_sim.py`)
Simulates proactive threat hunting scenarios:
- Hypothesis-driven hunting
- Baseline deviation detection
- Lateral movement detection
- Fileless malware indicators

**How to run:**
```bash
python threat_hunting_sim.py
```

---

## Learning Objectives

- Understand how to work with IOCs effectively
- Learn multi-source log correlation techniques
- Practice threat hunting methodologies

## Requirements

- Python 3.x (standard library only)
