# Projects: Secure Software Development

Hands-on projects for learning secure coding practices.

---

## Projects Included

### 1. Secure Code Examples (`secure_code_examples.py`)
Before/after code samples demonstrating secure coding:
- Input validation
- SQL injection prevention
- XSS prevention
- Secure authentication
- Cryptographic best practices

**How to run:**
```bash
python secure_code_examples.py
```

### 2. Threat Modeling Exercise (`threat_modeling.py`)
Interactive threat modeling simulation:
- STRIDE categorization
- Attack tree generation
- Mitigation planning
- Risk assessment

**How to run:**
```bash
python threat_modeling.py
```

---

## Learning Objectives

- Understand common coding vulnerabilities
- Learn to write secure code by example
- Practice threat modeling techniques

## Requirements

- Python 3.x (standard library only)
