# Projects: Emerging Technologies in Cybersecurity

Hands-on projects exploring security in emerging technologies.

---

## Projects Included

### 1. IoT Device Simulator (`iot_device_sim.py`)
Simulates IoT device security scenarios:
- Secure vs insecure device configurations
- MQTT communication simulation
- Device authentication models
- Firmware update security

**How to run:**
```bash
python iot_device_sim.py
```

### 2. Quantum Key Distribution Sim (`quantum_key_distribution.py`)
Simplified simulation of quantum cryptography:
- BB84 protocol demonstration
- Eavesdropping detection
- Key generation process
- Quantum vs classical security

**How to run:**
```bash
python quantum_key_distribution.py
```

### 3. Blockchain Security Demo (`blockchain_security.py`)
Demonstrates blockchain security concepts:
- Block hashing and chaining
- Tamper detection
- Consensus simulation
- Smart contract vulnerabilities

**How to run:**
```bash
python blockchain_security.py
```

---

## Learning Objectives

- Understand IoT security challenges
- Learn basics of quantum cryptography
- Grasp blockchain security principles

## Requirements

- Python 3.x (standard library only)
