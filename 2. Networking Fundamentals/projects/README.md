# Projects: Networking Fundamentals

Hands-on projects to understand networking concepts through code and simulation.

---

## Projects Included

### 1. Subnet Calculator (`subnet_calculator.py`)
A command-line tool that calculates:
- Network address
- Broadcast address
- Usable host range
- Number of hosts
- Subnet mask in binary and dotted-decimal

**How to run:**
```bash
python subnet_calculator.py
```

### 2. Port Scanner (`port_scanner.py`)
A simple TCP connect scanner that checks if common ports are open on a target host.

**How to run:**
```bash
python port_scanner.py
# Then enter target IP/hostname and port range
```

> **Note:** Only scan hosts you own or have explicit permission to scan.

### 3. DNS Lookup Tool (`dns_lookup_tool.py`)
Performs DNS resolution and reverse DNS lookups using Python's socket library.

**How to run:**
```bash
python dns_lookup_tool.py
```

---

## Learning Objectives

- Understand IP addressing and subnetting through calculation
- Learn how TCP port scanning works at a basic level
- Explore DNS resolution mechanics

## Requirements

- Python 3.x (standard library only)
