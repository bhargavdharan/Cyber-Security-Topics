# Projects: Operating Systems and Security

Hands-on projects to understand OS security concepts through simulations and analysis tools.

---

## Projects Included

### 1. File Permission Analyzer (`file_permission_analyzer.py`)
Analyzes and explains file permissions in Unix/Linux format (rwx). Includes:
- Permission string decoder
- Permission calculator (convert between numeric and symbolic modes)
- Security recommendations

**How to run:**
```bash
python file_permission_analyzer.py
```

### 2. System Hardening Checker (`system_hardening_checker.py`)
A simulated OS security audit tool that checks:
- Password policies
- Open ports
- Running services
- User account security
- Patch status

**How to run:**
```bash
python system_hardening_checker.py
```

### 3. Log Monitor (`log_monitor.py`)
Monitors log files for suspicious patterns and security events.

**How to run:**
```bash
python log_monitor.py
```

---

## Learning Objectives

- Understand file permission models
- Learn what makes an OS configuration secure or vulnerable
- Practice log analysis for security monitoring

## Requirements

- Python 3.x (standard library only)
