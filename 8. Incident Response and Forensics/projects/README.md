# Projects: Incident Response and Forensics

Hands-on simulations for learning incident response and digital forensics concepts.

---

## Projects Included

### 1. Log Incident Analyzer (`incident_analyzer.py`)
Simulates investigating a security incident through log analysis:
- Timeline reconstruction
- Attack pattern identification
- Indicator of Compromise (IOC) extraction
- Evidence correlation

**How to run:**
```bash
python incident_analyzer.py
```

### 2. File Integrity Checker (`file_integrity_checker.py`)
Demonstrates evidence integrity verification:
- Generate file hashes
- Detect tampering
- Create chain of custody records
- Compare file versions

**How to run:**
```bash
python file_integrity_checker.py
```

### 3. Memory Forensics Simulator (`memory_forensics_sim.py`)
Simulates analyzing a memory dump for malicious activity:
- Process enumeration
- Network connection analysis
- Injected code detection
- Suspicious pattern matching

**How to run:**
```bash
python memory_forensics_sim.py
```

---

## Learning Objectives

- Practice reconstructing attack timelines from logs
- Understand chain of custody and evidence integrity
- Learn how memory forensics detects advanced threats

## Requirements

- Python 3.x (standard library only)
