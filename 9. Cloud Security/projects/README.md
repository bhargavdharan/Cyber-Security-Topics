# Projects: Cloud Security

Hands-on simulations for understanding cloud security concepts and configurations.

---

## Projects Included

### 1. IAM Policy Simulator (`iam_policy_simulator.py`)
Simulates AWS-style IAM policy evaluation:
- Create users, groups, and roles
- Define policies with allow/deny rules
- Test access decisions
- Understand policy evaluation logic

**How to run:**
```bash
python iam_policy_simulator.py
```

### 2. S3 Bucket Security Scanner (`s3_security_scanner.py`)
Simulates checking cloud storage configurations:
- Public access detection
- Encryption verification
- Versioning status
- Logging configuration

**How to run:**
```bash
python s3_security_scanner.py
```

### 3. Cloud Configuration Auditor (`cloud_config_auditor.py`)
Audits simulated cloud infrastructure for security misconfigurations:
- Security group rules
- Encryption settings
- MFA enforcement
- Logging and monitoring

**How to run:**
```bash
python cloud_config_auditor.py
```

---

## Learning Objectives

- Understand IAM policy evaluation and least privilege
- Learn to identify risky cloud storage configurations
- Practice auditing cloud infrastructure for misconfigurations

## Requirements

- Python 3.x (standard library only)
