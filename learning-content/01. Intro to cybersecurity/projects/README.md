# Projects: Introduction to Cybersecurity

This folder contains hands-on projects and simulations to help you understand the fundamentals of cybersecurity.

---

## Projects Included

### 1. CIA Triad Simulator (`cia_triad_simulator.py`)
An interactive demonstration of the three core principles of cybersecurity:
- **Confidentiality:** Encrypt and decrypt messages using a Caesar cipher
- **Integrity:** Generate and verify file hashes to detect tampering
- **Availability:** Simulate a service uptime monitor with random failures

**How to run:**
```bash
python cia_triad_simulator.py
```

### 2. Password Strength Checker (`password_strength_checker.py`)
Analyzes passwords and provides feedback on strength based on:
- Length (minimum 8 characters)
- Presence of uppercase, lowercase, digits, and special characters
- Common password patterns
- Entropy estimation

**How to run:**
```bash
python password_strength_checker.py
```

### 3. Social Engineering Awareness Quiz (`social_engineering_quiz.py`)
A text-based quiz that tests your ability to recognize common social engineering tactics including phishing, pretexting, baiting, and tailgating scenarios.

**How to run:**
```bash
python social_engineering_quiz.py
```

---

## Learning Objectives

After completing these projects, you should be able to:
- Explain confidentiality, integrity, and availability with practical examples
- Evaluate password strength and understand what makes a password secure
- Identify common social engineering techniques

## Requirements

- Python 3.x (standard library only)
