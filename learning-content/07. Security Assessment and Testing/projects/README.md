# Projects: Security Assessment and Testing

Hands-on tools for learning vulnerability assessment and penetration testing concepts.

---

## Projects Included

### 1. Vulnerability Scanner (`vulnerability_scanner.py`)
Simulates a vulnerability assessment by checking for:
- Weak password policies
- Open risky ports
- Outdated software versions
- Missing security patches
- Common misconfigurations

**How to run:**
```bash
python vulnerability_scanner.py
```

### 2. Password Security Auditor (`password_auditor.py`)
Educational password security analysis tool:
- Dictionary attack simulation
- Brute force time estimation
- Password hash comparison
- Policy compliance checking

**How to run:**
```bash
python password_auditor.py
```

> **Warning:** This tool is for educational purposes only. Only test passwords you own.

### 3. Web Directory Enumerator (`dir_enumerator.py`)
Simulates web directory enumeration for finding hidden resources:
- Common path wordlist
- Status code simulation
- Rate limiting demonstration

**How to run:**
```bash
python dir_enumerator.py
```

---

## Learning Objectives

- Understand how vulnerability scanners identify weaknesses
- Learn why strong passwords matter through time estimates
- Practice reconnaissance techniques safely

## Requirements

- Python 3.x (standard library only)
