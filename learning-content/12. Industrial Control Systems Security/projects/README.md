# Projects: ICS Security

Hands-on simulations for understanding industrial control system security.

---

## Projects Included

### 1. ICS Network Simulator (`ics_network_sim.py`)
Simulates an industrial control network:
- Purdue model levels
- OT protocol simulation (Modbus, DNP3)
- Network segmentation visualization
- Traffic flow analysis

**How to run:**
```bash
python ics_network_sim.py
```

### 2. SCADA HMI Simulator (`scada_hmi_sim.py`)
Simulates a SCADA Human-Machine Interface:
- Process monitoring display
- Alarm management
- Control command validation
- Operator action logging

**How to run:**
```bash
python scada_hmi_sim.py
```

---

## Learning Objectives

- Understand ICS network architecture
- Learn OT protocol basics
- Practice SCADA security concepts

## Requirements

- Python 3.x (standard library only)
