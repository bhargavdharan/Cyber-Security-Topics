# Projects: Advanced Persistent Threats

Hands-on simulations for understanding APT tactics and detection.

---

## Projects Included

### 1. APT Lifecycle Simulator (`apt_lifecycle_sim.py`)
Simulates the complete APT kill chain:
- Reconnaissance through exfiltration
- MITRE ATT&CK technique mapping
- Timeline reconstruction
- Detection opportunities at each stage

**How to run:**
```bash
python apt_lifecycle_sim.py
```

### 2. Persistence Mechanism Detector (`persistence_detector.py`)
Simulates detecting APT persistence techniques:
- Registry run keys
- Scheduled tasks
- WMI event subscriptions
- Service-based persistence

**How to run:**
```bash
python persistence_detector.py
```

---

## Learning Objectives

- Understand the APT kill chain
- Learn common persistence mechanisms
- Practice detecting advanced threats

## Requirements

- Python 3.x (standard library only)
